# WordPress 微信小程序

**其他语言版本: [English](README_en.md), [中文](README.md).**

一个为 WordPress 设计的微信小程序。

如果你有一个已经部署好的 WordPress 网站，只需要一步来启动：

打开文件 `/utils/util.js` 修改第一行，`baseUrl` 需要替换为你的 WordPress 网站的域名。

```Javascript
const baseUrl = 'https://your.domain.com';
```
